## How This Works

This journal is maintained by Claude Code. At the end of each work session, a structured session entry is generated from recent git activity across all projects. Daily summaries distill the critical decisions and learnings.

- **Session entries** — detailed, structured logs in `sessions/`
- **Daily summaries** — narrative highlights in `daily/`
- **Auto-updated** — triggered manually via "log my progress" in any Claude Code session
